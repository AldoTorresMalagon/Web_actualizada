Opciones de Accesibilidad

Personaliza tu experiencia en nuestro sitio web según tus necesidades

Haciendo nuestro sitio accesible para todos

En Cafetería ITH nos preocupamos por brindar una experiencia inclusiva. Utiliza estas herramientas para adaptar el sitio web a tus necesidades visuales y de lectura. Los cambios se guardan automáticamente.

Tamaño del Texto

Ajusta el tamaño de la fuente según tu comodidad visual

Alto Contraste

Aumenta el contraste de colores para mejor visibilidad

Activar Alto Contraste
Modo Oscuro

Invierte los colores para reducir el cansancio visual

Activar Modo Oscuro
Modo Lectura

Fondo cálido que reduce la fatiga en lectura prolongada

Activar Modo Lectura
Fuente para Dislexia

Usa OpenDyslexic, diseñada para facilitar la lectura

Activar Fuente OpenDyslexic
Subrayar Enlaces

Subraya todos los enlaces para identificarlos más fácilmente

Subrayar Todos los Enlaces
Reducir Movimiento

Desactiva animaciones y transiciones (recomendado para epilepsia)

Desactivar Animaciones
Cursor Grande

Aumenta el tamaño del cursor para mayor visibilidad

Activar Cursor Grande
Resaltar Focus

Resalta el elemento activo al navegar con el teclado

Activar Indicador de Focus
Saturación de Color

Ajusta la intensidad de colores (útil para daltonismo)
